Packages for ArduCopter Mega board (written / compiled on Arduino 2.2 IDE)

ART_Quadrotor: C version
	light weight, easy to deploy.
	requires: APM_ADC, APM_BMP085, APM_Compass, APM_RC, GPS_NMEA libraries

ART_Arduino: C++ OOP version
	easy to understand and furder develop.
	requires: AP_ADC, AP_Math and APM_RC libraries
